# FoodTruckNerdz Workspace

This is the Base repository. It holds instructions and support files for setting up your local machine and committing code.

## Quick Start

1. Clone this repository to your machine.
2. Clone any other FoodTruckNerdz repositories you need to work on into the same parent directory.
3. Open the `FoodTruckNerdz.code-workspace` file in VSCode to launch the workspace.

## Documentation

For complete setup instructions, workspace configuration, and development guidelines, see the [Antora documentation](https://foodtrucknerds.github.io/docs).
