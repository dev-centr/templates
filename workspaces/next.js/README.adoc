= Next.js Template

The `next.js` template builds on the JavaScript baseline and adds editor helpers tuned for Next.js apps. Include it whenever you are working on Next.js 13+ projects that need modern file-based routing, server actions, and component snippets.

== Included extensions

All recommendations come from the community-run OpenVSX marketplace, so they install cleanly in Cursor and other VS Code forks:

* `nrjdalal.nextjs-assistant` – in-editor guidance for Next.js conventions.
* `marcbruederlin.next-icons` – Next.js-aware file icons for the VS Code explorer.
* `HareeshBhittam.next-js-snippets-latest` – snippets for the latest Next.js patterns (app router, server components, etc.).
* `kristiyanvelkov.nextjs-file-gelenator` – quick scaffolding for common Next.js file layouts.

== How to use it

To use this template, open the `nextjs.code-workspace` file in this folder and copy its `settings` and `extensions` blocks into your project's `.code-workspace` file.

[IMPORTANT]
====
For a complete setup, you should also copy the baseline settings and extensions from the `_common/common.code-workspace` file and merge them with these Next.js-specific ones.
====

Copying these blocks directly into your project ensures that your development environment is fully configured with the shared standards of the Dev Center, without needing to reference external folders.
