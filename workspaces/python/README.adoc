= Python Template

Use the `python` template when you want Ruff-based formatting and linting for API prototypes, data scripts, or automation tooling. It complements the `_common` template by adding the fast Ruff toolchain everywhere you edit Python files.

== Included extension

* `charliermarsh.ruff` – Formatter, linter, and import organizer powered by Ruff. Published on OpenVSX so it works in Cursor and other VS Code forks.

== Recommended workspace composition

== How to use it

To use this template, open the `python.code-workspace` file in this folder and copy its `settings` and `extensions` blocks into your project's `.code-workspace` file.

[IMPORTANT]
====
For a complete setup, you should also copy the baseline settings and extensions from the `_common/common.code-workspace` file and merge them with these Python-specific ones.
====

Copying these blocks directly into your project ensures that your development environment is fully configured with the shared standards of the Dev Center, without needing to reference external folders.
