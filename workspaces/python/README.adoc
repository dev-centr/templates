= Python Template

Use the `python` template when you want Ruff-based formatting and linting for API prototypes, data scripts, or automation tooling. It complements the `_common` template by adding the fast Ruff toolchain everywhere you edit Python files.

== Included extension

* `charliermarsh.ruff` – Formatter, linter, and import organizer powered by Ruff. Published on OpenVSX so it works in Cursor and other VS Code forks.

== Recommended workspace composition

== How to use it

To use this template, copy `.vscode/settings.json` and `.vscode/extensions.json` from this folder into your project (merge with any existing `.vscode` files).

[IMPORTANT]
====
For a complete setup, merge in the `.vscode` files from `workspaces/_common/` as well.
====

Copying or merging these files into your project ensures that your development environment is fully configured with the shared standards of the Dev Center, without needing to reference external folders.
