= Astro Template

The `astro` template provides a specialized environment for building modern, content-driven websites with the Astro framework. It includes optimized settings for `.astro` files, MDX support, and component-based development.

== Included extensions

All recommendations come from the community-run OpenVSX marketplace, ensuring compatibility with Cursor and other VS Code forks:

* `astro-build.astro-vscode` – Official Astro support, including syntax highlighting, IntelliSense, and diagnostics.
* `biomejs.biome` – Fast, unified tool for formatting and linting JavaScript, TypeScript, and CSS.
* `unifiedjs.vscode-mdx` – Language support for MDX, commonly used in Astro for content-rich pages.

== How to use it

To use this template, copy `.vscode/settings.json` and `.vscode/extensions.json` from this folder into your project (merge with any existing `.vscode` files).

[IMPORTANT]
====
For a complete setup, merge in the `.vscode` files from `workspaces/_common/` as well.
====

Including these settings adds Astro-specific diagnostics and snippets while maintaining the shared standards from the `_common` template.

== Features

* **Tailored Formatting**: Pre-configured settings for `.astro` files to ensure consistent code style.
* **Content Focus**: First-class support for MDX and Markdown, essential for Astro's "Content Collections".
* **Performance**: Workspace excludes for `dist/` and `node_modules/` to keep search and the explorer snappy.
