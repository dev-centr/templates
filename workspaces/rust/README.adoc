= Rust Template

The `rust` template provides shared Rust developer tooling configuration. Including this folder in workspaces ensures consistent extension recommendations and deprecations across all Rust projects.

NOTE: Only extensions available on https://open-vsx.org[OpenVSX] are recommended. Extensions not published to OpenVSX must not be added to this template, in order to be compatible with non-Microsoft-owned forks of VSCode.

== Current status

* Recommends `rust-lang.rust-analyzer` (the actively maintained Rust extension, available on OpenVSX).
* Deprecates `rust-lang.rust` (the legacy extension, no longer maintained).
* You can include this folder in `.code-workspace` files to pick up shared tooling automatically.

== How to use it

To use this template, open the `rust.code-workspace` file in this folder and copy its `settings` and `extensions` blocks into your project's `.code-workspace` file.

[IMPORTANT]
====
For a complete setup, you should also copy the baseline settings and extensions from the `_common/common.code-workspace` file and merge them with these Rust-specific ones.
====

Copying these blocks directly into your project ensures that your development environment is fully configured with the shared standards of the Dev Center, without needing to reference external folders.

Any project that includes these workspace blocks will pick up the extension recommendations and deprecations. Before adding new extensions, verify they are available on https://open-vsx.org[OpenVSX].
