= Rust Template

The `rust` template provides shared Rust developer tooling configuration. Including this folder in workspaces ensures consistent extension recommendations and deprecations across all Rust projects.

NOTE: Only extensions available on https://open-vsx.org[OpenVSX] are recommended. Extensions not published to OpenVSX must not be added to this template, in order to be compatible with non-Microsoft-owned forks of VSCode.

== Current status

* Recommends `rust-lang.rust-analyzer` (the actively maintained Rust extension, available on OpenVSX).
* Deprecates `rust-lang.rust` (the legacy extension, no longer maintained).
* You can copy this folder's `.vscode` files into your repository to pick up shared tooling.

== How to use it

To use this template, copy `.vscode/settings.json` and `.vscode/extensions.json` from this folder into your project (merge with any existing `.vscode` files).

[IMPORTANT]
====
For a complete setup, merge in the `.vscode` files from `workspaces/_common/` as well.
====

Copying or merging these files into your project ensures that your development environment is fully configured with the shared standards of the Dev Center, without needing to reference external folders.

Any project that includes this configuration will pick up the extension recommendations and deprecations. Before adding new extensions, verify they are available on https://open-vsx.org[OpenVSX].
