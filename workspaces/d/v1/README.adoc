= D Template

The `d` template provides shared D language developer tooling configuration. Including this folder in workspaces ensures consistent extension recommendations across all D projects.

NOTE: Only extensions available on https://open-vsx.org[OpenVSX] are recommended. Extensions not published to OpenVSX must not be added to this template.

== Current status

* Recommends `webfreak.code-d` (the actively maintained D extension, available on OpenVSX).
* You can copy this folder's `.vscode` files into your repository to pick up shared tooling.

== How to use it

To use this template, copy `.vscode/settings.json` and `.vscode/extensions.json` from this folder into your project (merge with any existing `.vscode` files).

[IMPORTANT]
====
For a complete setup, merge in the `.vscode` files from `workspaces/_common/` as well.
====

Copying these blocks directly into your project ensures that your development environment is fully configured with the shared standards of the Dev Center.
